# _Wordle

A Wordle variant with 9 simultaneous boards, a space twist, and a decoy board.

## Features
- **9 boards** — guess all words simultaneously
- **Space twist** — some answers are 5-letter words with a leading space
- **Decoy board** — one board is fake and can never be solved
- **17 guesses** to solve all 8 real boards
- Live stats: boards solved, alphabet coverage, guess count

## Development

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub and connect to [Vercel](https://vercel.com) for instant deployment.
